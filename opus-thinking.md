## 1. Резюме проекта

**МТМ МОТО** — новый российский бренд мототехники, предлагающий квадроциклы, снегоходы, мотоциклы, питбайки, киткомплекты и мотоэкипировку. Бренд зарегистрирован, товарный знак и авторские права закреплены за командой проекта.

### Ключевые цифры

| Показатель | Значение |
|---|---|
| Объём инвестиций | 30 000 000 ₽ |
| Доля инвестора | 50% чистой прибыли |
| Срок соглашения | до 7 лет |
| Прогнозная чистая прибыль (Год 1) | **20 644 000 ₽** |
| Доход инвестора (Год 1) | **10 322 000 ₽** |
| Срок окупаемости | **~2 года 11 месяцев** |
| ROI за 7 лет | **376%** |
| Общий доход инвестора за 7 лет | **112 800 000 ₽** |

> **Главное преимущество для инвестора:** деньги вложены не в стены, а в ликвидный товар. При необходимости инвестиции могут быть возвращены через реализацию складских остатков без существенных потерь.

## 2. О компании и бренде

### Текущий статус

- **Бренд МТМ МОТО** — зарегистрированный товарный знак, все права закреплены за командой
- **Дилерский договор** — право на официальный ввоз техники «в белую» с ПСМ (паспорт самоходной машины)
- **Проработанная база поставщиков** — согласованные условия закупа, логистика, выгодные цены
- **Технически доработанные модели** — прошли испытания, подтвердили эффективность
- **Действующий магазин на Ozon** — топовые позиции по продажам мотоэкипировки
- **CRM-система** с первичной клиентской базой и начальными прозвонами по оптовым продажам
- **Готовые каналы продаж:** сайт, группа ВК, Telegram-канал, Ozon, прайсы для оптовиков
- **Укомплектованная команда** (5 человек), готовая к переезду в Пермь

### Предыдущий опыт

Команда ранее запускала проект МТМ МОТО в Кирове с привлечением инвесторов. За время работы:

- Отработаны каналы трафика (розница + опт)
- Проданы предоплатами 50% контейнера ещё до полной поставки
- Налажены бизнес-процессы открытия и ведения мотосалона
- Протестирован и подтверждён спрос на модельный ряд

Проект перезапускается в Перми по причине нечестного сотрудничества со стороны предыдущих инвесторов (расторжение по инициативе команды).

## 4. Анализ рынка

### 4.1. Общая ситуация

Российский рынок мототехники переживает период активного роста:

- **Уход западных брендов** (2022–2024) открыл ниши для новых игроков
- **Рост импорта китайской техники** — Россия стала крупнейшим рынком для китайских производителей мото
- **Тренд на импортозамещение** — потребители готовы к российским брендам с адекватным соотношением цена/качество
- **Снижение конкуренции** в сегменте «до 500 000 ₽» — основные японские/европейские бренды ушли в премиум-сегмент

### 4.2. Объём рынка

| Сегмент | Оценка рынка РФ (2025) | Рост |
|---|---|---|
| Мотоциклы | ~120 000 ед./год | +15–20% г/г |
| Квадроциклы | ~45 000 ед./год | +10–15% г/г |
| Снегоходы | ~25 000 ед./год | +8–12% г/г |
| Питбайки | ~30 000 ед./год | +20–25% г/г |
| Мотоэкипировка | ~15 млрд ₽/год | +12–18% г/г |

### 4.3. Почему Пермь?

| Фактор | Значение |
|---|---|
| Население Пермского края | ~2,6 млн человек |
| Удалённость от конкурентов | Низкая плотность мотосалонов |
| Климат | Длинная зима → спрос на снегоходы + лето → мототехника |
| Логистическая доступность | Хаб для Урала и Приволжья |
| Уровень доходов | Соответствует бюджетному/среднему сегменту техники |

### 4.4. География поставок

Основные регионы сбыта (с учётом логистики):

- Пермский край
- Кировская область
- Республика Коми
- Удмуртская Республика
- Республика Марий Эл
- Свердловская область
- Другие регионы РФ (опт + доставка ТК)

## 13. Условия для инвестора

### 13.1. Структура сделки

| Параметр | Условие |
|---|---|
| Сумма инвестиций | 30 000 000 ₽ |
| Доля инвестора | 50% от чистой прибыли |
| Срок соглашения | до 7 лет |

| Начало выплат | С 1-го месяца получения прибыли |
| Периодичность выплат | Ежемесячно или ежеквартально (по согласованию) |

### 13.2. Гарантии и защита инвестора

| Гарантия | Описание |
|---|---|
| **Ликвидный товар** | 80%+ инвестиций — в товаре, который может быть продан в любой момент |
| **Прозрачная отчётность** | CRM-система (Битрикс24), ежемесячные P&L-отчёты, доступ к 1С |
| **Залог товарных запасов** | Товар на складе выступает обеспечением инвестиций |
| **Контроль** | Инвестор получает доступ к управленческой отчётности в реальном времени |
| **Диверсификация** | Множество товарных категорий, географий, каналов сбыта |

### 13.3. Преимущества данного формата

1. **Быстрый возврат** — деньги вложены в оборотный товар, не в стены и оборудование
2. **Отсутствие «замороженных» активов** — при форс-мажоре товар можно реализовать по себестоимости в течение 1–3 месяцев
3. **Масштабируемость** — рост прибыли инвестора пропорционален росту бизнеса
4. **Отсутствие субсидиарной ответственности** — инвестор не несёт операционных рисков
5. **Ограниченный срок** — через 7 лет обязательства перед инвестором прекращаются
